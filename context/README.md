# context/

Ephemeral notes. Session-specific scratch. Things that are true *right now* but not worth promoting to `projects/mosque-connect/` or `memory/`.

Examples:
- Working notes for an in-flight migration
- Temporary investigation findings
- Current-task state Claude needs to remember within a session

**Not for**: decisions (→ `DECISIONS.md`), permanent rules (→ `memory/`), project facts (→ `projects/mosque-connect/`).

Files here should be considered disposable.
