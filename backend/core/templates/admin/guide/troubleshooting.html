Troubleshooting

Something not working? Find solutions to common issues here.

I Can't Log In

Wrong password or email

Make sure you're using the email address you registered with, not a username. Check for typos and make sure Caps Lock is off. Passwords are case-sensitive.

If you've forgotten your password, ask a Super Admin to reset it for you from the Users section.

Account locked or disabled

If your account has been deactivated, you won't be able to log in. Contact the Super Admin or the person who manages the admin panel to reactivate your account.

Changes Not Showing in the App

I updated an announcement/event but the app still shows the old version

The app caches data to work offline. Community members may need to:

  • Pull to refresh — swipe down on the relevant screen in the app
  • Wait a few minutes — the app refreshes automatically every few minutes
  • Restart the app — close it completely and reopen

Your changes are saved immediately on the server. The delay is only on the app side.

New prayer times aren't appearing

Check that you saved the prayer times for the correct date and correct mosque. Go back to the prayer times list and verify the entry exists. If it does, the app should pick it up within a few minutes.

Errors When Saving

"This field is required" error

A required field is empty. Scroll through the form and look for fields highlighted in red. Fill in all required fields and try saving again. Required fields are marked — check the Quick Reference tables in each guide section for details.

"Mosque prayer time with this Mosque and Date already exists"

You're trying to add prayer times for a date that already has an entry. Go back to the prayer times list, find the existing entry for that date, and edit it instead of creating a new one.

"Enter a valid date/time" error

Make sure dates are in YYYY-MM-DD format (e.g., 2026-03-18) and times are in HH:MM format (e.g., 13:30). Don't include AM/PM — use 24-hour time.

Gift Aid Issues

CSV/XML export is empty

Make sure the Gift Aid Claim has donations assigned to it. Open the claim and check the "Donations" section — you need to select donations from the "Available" list and move them to the "Chosen" list. Only donations marked as "Gift Aid eligible" will appear.

"Please select exactly one claim" error on XML export

XML export works for one claim at a time. On the claims list, tick only one claim's checkbox before selecting the export action.

Charity Gift Aid Settings won't save

Make sure all required fields are filled in — especially the Charity Name, HMRC Reference, and Authorised Official details. If you see an error about the "Add" button being greyed out, settings already exist — click on the existing entry to edit it instead.

Push Notification Issues

I posted an announcement but nobody received a notification

Check these common causes:

  • No subscribers yet — community members need to subscribe to your mosque in the app first. Check the Subscriptions section to see if anyone is subscribed.
  • Notifications turned off — subscribers may have turned off announcement notifications in their app settings.
  • Phone settings — the person may have turned off notifications for the app in their phone's settings (Settings → Notifications).
  • Editing, not creating — notifications are only sent for new announcements. If you edited an existing one, no notification was sent.
Can I test if push notifications are working?

Ask your technical support team to run the test notification command. They can send a test notification to any specific user to verify the system is working. If you're the technical admin, run: python manage.py test_push your@email.com

Display Issues

The page looks broken or things are overlapping

Try these steps:

  • Refresh the page — press Ctrl+R (or Cmd+R on Mac)
  • Clear your browser cache — press Ctrl+Shift+Delete
  • Try a different browser — Chrome, Safari, or Firefox all work well
  • Check your internet connection — a slow connection can cause pages to load partially
The admin panel is very slow

This is usually a network issue. Check your internet speed. If the problem persists, it could be a server issue — contact technical support.
