Managing Events & Lessons

Add classes, lectures, youth programs, and community gatherings.

How to Add an Event

  1. Click Events in the left sidebar under "Content".
  2. Click "Add Event" in the top-right corner.
  3. Select your Mosque from the dropdown.
  4. Enter a clear Title (e.g., "Weekly Tafseer Class" or "Youth Football Tournament").
  5. Choose a Category — this helps the app organise events for community members:
    • Lesson — regular classes (fiqh, aqeedah, Arabic)
    • Lecture — one-off or special talks
    • Quran School — Quran circles, hifz classes, tajweed
    • Youth — youth-specific programs
    • Sisters — sisters-only events
    • Community — iftars, open days, fundraisers, etc.
  6. Set the Event Date, Start Time, and End Time.
  7. If this event happens regularly, set Recurring to "Weekly" or "Monthly".
  8. Add the Speaker name if applicable (optional).
  9. Add a Description with any extra details — who it's for, what to bring, etc.
  10. Click "Save".
Push notifications are sent automatically. When you save a new event, everyone subscribed to your mosque who has event notifications turned on will receive a push notification with the event name and date.
For recurring events (e.g., a weekly class every Saturday), you only need to create it once and set it to "Weekly". The app will show it each week automatically. The push notification is only sent once when the event is first created.

Event Categories Explained

Category Best For Examples
Lesson Regular educational classes Fiqh class, Arabic grammar, Kitab study
Lecture One-off talks, guest speakers Friday reminder, visiting scholar lecture
Quran School Quran-specific learning Tajweed class, hifz circle, Quran study
Youth Programs for young people Youth halaqah, sports day, mentoring
Sisters Sisters-only programs Sisters' class, support circle
Community General community gatherings Iftar, charity dinner, open day

Editing or Deleting an Event

  1. Click Events in the sidebar to see all events.
  2. Click the title of the event you want to change.
  3. Make your edits and click "Save", or scroll down and click "Delete" to remove it.
You can filter events by category, recurring type, or mosque using the filters on the right side of the events list.

Quick Reference
