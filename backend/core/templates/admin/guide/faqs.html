Frequently Asked Questions

Quick answers to common questions about managing the admin panel.

General

How do I add a new mosque admin?

Go to Mosque Admins in the sidebar, click "Add Mosque Admin", select the person's user account and your mosque, choose their role (Admin or Super Admin), and save. The person must already have a user account — ask them to sign up in the app first.

What's the difference between Admin and Super Admin?

Admin can create and edit announcements, events, and prayer times for their mosque.

Super Admin can do everything above, plus manage other admins, view donation records, and submit Gift Aid claims. Give this role only to trusted committee members.

Can I schedule an announcement for the future?

Announcements are published immediately when you create them. However, you can set an Expiry Date so they automatically disappear after a certain date. If you need to prepare an announcement in advance, save it as a draft by noting the details elsewhere and creating it on the day you want it to go live.

How do I change the mosque photo?

Go to Mosques in the sidebar, click on your mosque, scroll to the Photo field, and upload a new image. Use a high-quality photo of the mosque exterior — it will be shown in the app to community members.

Prayer Times

What does "calculation method" mean?

The calculation method determines how prayer start times are calculated based on sun angles. The default is Umm Al-Qura (used in Saudi Arabia). This affects the automatically calculated start times, not the jama'ah times you set manually. You generally don't need to change this.

How often should I update prayer times?

Update jama'ah times whenever they change at your mosque — usually seasonally (every few weeks or months), during Ramadan, for Eid, and when clocks change. Community members rely on these being accurate, so keeping them up to date is important.

Do I need to set prayer start times?

No. Prayer start times (Fajr begins, Dhuhr begins, etc.) are calculated automatically based on your mosque's location. You only need to set the jama'ah times — when the congregation actually prays.

Announcements & Events

I accidentally deleted an announcement — can I get it back?

Unfortunately, deleted announcements cannot be recovered. You'll need to create a new one. To avoid this, consider setting an expiry date instead of deleting — expired announcements simply stop showing in the app.

How long do announcements stay visible?

If you set an Expiry Date, the announcement will automatically disappear after that date. If you don't set one, it stays visible until you manually delete it.

Can I create an event that happens every week?

Yes! When creating an event, set the Recurring field to "Weekly" or "Monthly". You only need to create the event once — the app will automatically show it on the appropriate days.

Donations & Gift Aid

How do donors receive receipts?

You can generate receipts from the admin panel. Go to Donations, select the donation(s), and use the "Generate donation receipt" action. Receipts include all details including Gift Aid information where applicable. You can then print or save the receipt as a PDF to share with the donor.

How do I submit a Gift Aid claim to HMRC?

1. Make sure Charity Gift Aid Settings are filled in (your charity name, HMRC reference, authorised official).

2. Go to Gift Aid Claims and create a new claim with the period dates.

3. Add the eligible donations to the claim.

4. Save, then use the Actions dropdown to export as CSV or XML.

5. Upload the file on the HMRC Charities Online portal.

Why can't I see the donations section?

The donations section is only visible to Super Admins and staff users. If you need access to donations and Gift Aid, ask a Super Admin to upgrade your role, or contact the person who manages the admin panel.

Do I need to manually add donations?

No — donations made through the website are recorded automatically via Stripe. You only need to add donations manually if someone donates via bank transfer, cash, or other offline methods and you want to track it in the system.

Push Notifications

Do I need to send notifications manually?

No — push notifications are sent automatically when you create a new announcement or event. You don't need to do anything extra. Just save the announcement or event and subscribers will be notified.

Who receives the push notification?

Only community members who have:

  • Subscribed to your mosque in the app
  • Turned on the relevant notification type (announcements or events) in their app settings
  • Allowed notifications on their phone

You can see subscriber preferences in the Subscriptions section of the admin panel.

Does editing an announcement send another notification?

No. Push notifications are only sent when an announcement or event is first created. Editing an existing one does not send another notification. If you need to re-notify people, delete the old announcement and create a new one.

What happens when someone uninstalls the app?

Their push token becomes invalid. The system automatically detects this and removes the old token so it doesn't slow things down. You don't need to do anything.

Account & Access

I forgot my password — how do I reset it?

On the admin login page, look for the "Forgot password?" link. If you don't see one, contact the Super Admin or technical support to reset your password manually.

Can I use this admin panel on my phone?

Yes! The admin panel works on mobile browsers. Open your phone's browser and go to the admin URL. The layout will adjust to fit your screen. For the best experience, use a tablet or desktop.
